- Main features: English and Spanish vocabulary practice, listening/writing, Czech-to-target translation, speech recognition, built-in vocabulary, unicorn UI, progress overview, flower reward video.

## Local upload

The full Xcode source package contains binary assets and video files. Push from Mac with:

```bash
git clone https://github.com/JaroslavArbo/English_For_Emily.git
cd English_For_Emily
cp -R /path/to/EnglishForEmily_complete_project/* .
git add .
git commit -m "Add English for Emily iOS source"
git push origin main
```
